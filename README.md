# 🎮 Мини-игры

Интерактивный сайт с набором из 6 мини-игр на чистом HTML, CSS и JavaScript.

## 🚀 Демо

Откройте `index.html` в браузере — всё работает без сборки и сервера.

## 🎯 Игры

| # | Название | Описание |
|---|----------|----------|
| 1 | **Угадай число** | Компьютер загадывает число от 1 до 100, нужно угадать за минимум попыток |
| 2 | **Простая арифметика** | Решай случайные примеры на сложение, вычитание, умножение и деление |
| 3 | **Переверни текст** | Введи текст — получи его в обратном порядке |
| 4 | **Камень, ножницы, бумага** | Классическая игра против компьютера |
| 5 | **Простая викторина** | Отвечай на вопросы с вариантами ответов |
| 6 | **Генератор случайных цветов** | Меняй фон секции на случайный цвет одним кликом |

## ✨ Особенности

- **Модальные окна** — все игры работают через кастомные модальные окна (без `alert`/`prompt`/`confirm`)
- **Адаптивная вёрстка** — от мобильных (до 860px) до десктопа
- **Анимации** — ховер-эффекты на карточках, плавные переходы, бегущая строка
- **Декоративные элементы** — SVG-звёзды на фоне (разные для десктопа и мобильных)
- **Плавный скролл** — кнопка «Поехали!» ведёт к блоку с играми

## 🛠 Технологии

- HTML5
- CSS3 (Flexbox, Grid, анимации, медиа-запросы)
- Vanilla JavaScript (ES5/ES6)
- Google Fonts — Montserrat


| Брейкпоинт | Особенности |
|------------|-------------|
| `> 1280px` | Полная версия, 3 колонки карточек |
| `≤ 1280px` | Карточки переносятся, гибкая сетка |
| `≤ 1024px` | Игры в колонку, уменьшенные шрифты |
| `≤ 860px`  | Мобильная версия: скрыт блок «Об играх», картинки сверху, мобильные звёзды |
